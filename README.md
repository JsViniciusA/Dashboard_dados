# Dashboard de Vendas — Season Pass Xbox

Descrição
---------
Este repositório contém uma dashboard em arquivo Excel com análise das vendas do Season Pass do Xbox ao longo do ano (2024). A dashboard reúne dados brutos, métricas-chave e visualizações interativas (gráficos e slicers) para apoiar decisões e apresentar a evolução de vendas por período e outras dimensões relevantes.

Arquivo incluído
----------------
- Dashboard.xlsx

O que há na dashboard
---------------------
- Dados brutos: planilha com o conjunto original de vendas (ex.: data da venda, produto, quantidade, preço, região, canal, ID do cliente).
- Dashboard: visualizações interativas com KPIs e gráficos (ex.: total de vendas, receita, vendas por mês).
- Métricas / Tabelas auxiliares: cálculos e tabelas dinâmicas que alimentam os gráficos (médias, percentuais).
- Gráficos: barras comparativas e outras visualizações.

Principais métricas exibidas
---------------------------------------
- Receita total do Season Pass no período anual
- Número total de unidades vendidas
- Vendas por mês (sazonalidade)
- Top N produtos / pacotes

Como abrir e usar
-----------------
1. Abra o arquivo com Microsoft Excel (recomendado: Excel 2016 ou superior / Microsoft 365).
2. Caso existam macros, ative as macros apenas se confiar na origem do arquivo.
3. Navegue até a aba `Dashboard` e utilize os filtros/slicers para interagir com as visualizações.
4. Para editar cálculos, verifique as abas de suporte (ex.: `Cálculos`, `Bases` e tabelas dinâmicas).

Ferramentas compatíveis
-----------------------
- Microsoft Excel (recomendado)
- LibreOffice / OpenOffice: pode abrir o arquivo, mas algumas funcionalidades (slicers, Power Query, macros e formatações avançadas) podem não funcionar corretamente.

Boas práticas e notas
---------------------
- Mantenha uma cópia dos dados brutos antes de editar.
- Documente qualquer alteração nas fórmulas ou na modelagem dos dados.
- Se for atualizar dados periodicamente, padronize o formato de importação (colunas, tipos e nomes).
